Dropdown

Dropdown adalah komponen interaktif yang menampilkan daftar menu atau opsi ketika pengguna mengklik tombol atau elemen toggle. VFUI menyediakan dropdown yang elegan dan mudah digunakan.

Variants

Class Deskripsi
.ui-dropdown Container utama dropdown
.ui-dropdown__toggle Elemen toggle untuk membuka dropdown
.ui-dropdown__menu Menu container yang berisi item-item
.ui-dropdown__item Item dalam dropdown
.ui-dropdown__item--active Item yang sedang aktif/dipilih
.ui-dropdown__divider Divider/separator antar item
.ui-dropdown--active State aktif dropdown (menu terlihat)
.ui-dropdown--right Dropdown dengan alignment kanan

JavaScript Options

Dropdown menggunakan deklarasi data-module="dropdown" untuk inisialisasi otomatis.

Events

Event Deskripsi
dropdown:select Di-trigger ketika item dipilih, berisi detail.value dari item

Contoh Event Handling

document.querySelector('.ui-dropdown').addEventListener('dropdown:select', function(e) {
  console.log('Selected value:', e.detail.value);
});
