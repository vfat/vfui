Button

Button adalah komponen interaktif yang digunakan untuk tindakan pengguna. VFUI menyediakan berbagai variant dan ukuran button yang dapat digunakan dengan mudah.

Variants

Primary Button

Preview
<button class="ui-button ui-button--primary">Primary</button>

Secondary Button

Preview
<button class="ui-button ui-button--secondary">Secondary</button>

Outline Button

Preview
<button class="ui-button ui-button--outline">Outline</button>
<button class="ui-button ui-button--outline-primary">Outline Primary</button>

Ghost Button

Preview
<button class="ui-button ui-button--ghost">Ghost</button>

State Variants

Preview
<button class="ui-button ui-button--primary" disabled>Disabled</button>
<button class="ui-button ui-button--primary ui-button--loading">Loading</button>

Sizes

Preview
<button class="ui-button ui-button--primary ui-button--sm">Small</button>
<button class="ui-button ui-button--primary">Default</button>
<button class="ui-button ui-button--primary ui-button--lg">Large</button>
<button class="ui-button ui-button--primary ui-button--xl">Extra Large</button>

Button Groups

Preview

<div class="ui-button-group">
  <button class="ui-button ui-button--outline">One</button>
  <button class="ui-button ui-button--outline">Two</button>
  <button class="ui-button ui-button--outline">Three</button>
</div>

Icon Buttons

Preview

  <svg>...</svg>
</button>

Variants Reference

Class Description
ui-button--primary Primary action button dengan warna utama
ui-button--secondary Secondary action button dengan warna biru
ui-button--outline Button dengan border, tanpa background
ui-button--outline-primary Outline button dengan border primer
ui-button--ghost Button transparan dengan hover effect
ui-button--danger Button untuk aksi berbahaya (delete)
ui-button--success Button untuk aksi sukses

Size Reference

Class Description
ui-button--sm Small button
ui-button--lg Large button
ui-button--xl Extra large button
ui-button--icon-only Square button untuk icons
