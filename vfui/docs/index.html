VFUI Framework

VFUI adalah framework UI modern yang ringan, tanpa dependensi eksternal, dan dibangun dengan HTML, CSS, dan JavaScript murni. Dirancang untuk developer yang membutuhkan fleksibilitas penuh dengan footprint yang minimal.

Fitur Utama

  • Zero Dependencies - Tidak ada library eksternal, build tools, atau preprocessors
  • Modular - Pilih dan gunakan hanya komponen yang diperlukan
  • Mobile-First - Dirancang untuk responsif di semua ukuran layar
  • Accessible - Mendukung WCAG 2.1 AA dengan focus states dan ARIA
  • Easy to Use - Declarative initialization dengan data-module attribute

Quick Start

Gunakan VFUI dengan menambahkan CSS dan JS files ke HTML:

<!-- CSS -->
      <link rel="stylesheet" href="vfui/css/variables.css">
      <link rel="stylesheet" href="vfui/css/base.css">
      <link rel="stylesheet" href="vfui/css/core.css">
      <link rel="stylesheet" href="vfui/css/button.css">

      <!-- JavaScript -->
      <script src="vfui/js/core.js"></script>
      <script src="vfui/js/modules/accordion.js"></script>
      <script src="vfui/js/modules/modal.js"></script>

Modal

Preview
<!-- Trigger -->
<button data-modal-trigger="my-modal">Open Modal</button>

<!-- Modal Element -->
<div class="ui-modal" id="my-modal" data-module="modal">
  <div class="ui-modal__backdrop"></div>
  <div class="ui-modal__container">
    <div class="ui-modal__header">
      <h3 class="ui-modal__title">Modal Title</h3>
      <button class="ui-modal__close">×</button>
    </div>
    <div class="ui-modal__body">
      Modal content here
    </div>
    <div class="ui-modal__footer">
      <button class="ui-button ui-button--outline" data-modal-close>Cancel</button>
      <button class="ui-button ui-button--primary">Confirm</button>
    </div>
  </div>
</div>
